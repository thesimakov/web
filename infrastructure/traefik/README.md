## Traefik (локальная маршрутизация)

В dev-режиме Traefik принимает HTTP на `:80` и роутит по Host‑правилам к контейнерам сайтов/сервисов.

### Домен для локальной разработки

Используйте wildcard домен (например `*.lmnt.local`) и добавьте нужные записи в `/etc/hosts`, например:

```
127.0.0.1 dashboard.lmnt.local
127.0.0.1 api.lmnt.local
```

Для сгенерированных сайтов — `project-slug.lmnt.local`.
